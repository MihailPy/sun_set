# Changelog

Все заметные изменения проекта документируются в этом файле.

## [1.0.0] - 2026-07-16

### Добавлено

- Создание и сохранение проектов с городами.
- Расчёт данных о времени заката.
- Просмотр и ручное редактирование данных заката.
- Статусы актуальности данных города.
- Экспорт данных городов в изображения.
- Предпросмотр экспортируемых изображений.
- Пользовательская справка внутри приложения.
- Окно «О программе».
- Сборки для macOS и Windows.

### Улучшено

- Интерфейс таблицы городов.
- Обработка повреждённых файлов проектов.
- Обработка ошибок сохранения.
- Проверка файлов настроек экспорта.
- Пользовательские сообщения об ошибках.

### Технические изменения

- Разделение логики на сервисы и workflow.
- Упрощение `MainWindow` и `CityTableModel`.
- Интеграционные тесты сохранения и загрузки проектов.
- Автоматические сборки через GitHub Actions.

### Fixed

- Fixed image export settings file dialogs.
- Fixed crash when saving image export settings.
