# Руководство пользователя Sun Set

## Назначение приложения

Sun Set позволяет создавать проекты с городами, рассчитывать данные о времени
заката, редактировать их и экспортировать изображения.

## Создание проекта

1. Запустите приложение.
2. Добавьте один или несколько городов.
3. Укажите год и дополнительные параметры проекта.
4. Сохраните проект через меню «Файл».

## Добавление города

Опишите, какие поля нужно заполнить:

- название;
- регион;
- широта;
- долгота;
- часовой пояс;
- высота над уровнем моря.

## Получение данных заката

1. Выберите города в таблице.
2. Нажмите «Обновить».
3. Дождитесь завершения расчёта.
4. Проверьте колонку «Статус».

## Статусы

- **Актуально** — данные рассчитаны для текущих параметров города.
- **Требует обновления** — координаты или другие параметры города были изменены.
- **Изменено вручную** — данные заката редактировались пользователем.
- **Ошибка расчёта** — данные не удалось рассчитать.
- **Нет данных** — расчёт ещё не выполнялся.

## Просмотр и редактирование данных

Нажмите «Открыть» в колонке «Данные заката».

## Экспорт изображения

1. Укажите файл настроек экспорта.
2. Укажите папку для сохранения.
3. Выберите города.
4. Используйте предпросмотр или экспорт.

## Работа с проектами

Проекты сохраняются в формате JSON.

Не редактируйте файл вручную без необходимости.

## Возможные ошибки

Опиши:

- повреждённый файл проекта;
- отсутствующий файл настроек;
- неправильные параметры экспорта;
- ошибки записи файла.
