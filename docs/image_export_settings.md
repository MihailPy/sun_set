# Image export settings

Файл настроек управляет фоном, шрифтом и расположением текста на изображении.

## Пример

```json
{
  "image": {
    "width": 1000,
    "height": 1400,
    "background_color": "#ffffff",
    "template_path": null
  },
  "text": {
    "font_path": null,
    "font_size": 24,
    "color": "#000000"
  },
  "layout": {
    "row_height": 32,
    "first_column_offset_x": 10,
    "second_column_offset_x": 130,
    "month_blocks": {
      "1": {"x": 40, "y": 60},
      "2": {"x": 360, "y": 60},
      "3": {"x": 680, "y": 60}
    }
  }
}
```

## Поля

`image`

- `width` — ширина изображения, если шаблон не используется.
- `height` — высота изображения, если шаблон не используется.
- `background_color` — цвет фона, если шаблон не используется.
- `template_path` — путь к шаблону. Если `null`, используется однотонный фон.

`text`

- `font_path` — путь к `.ttf` шрифту. Если `null`, используется стандартный шрифт.
- `font_size` — размер текста.
- `color` — цвет текста.

`layout`

- `row_height` — расстояние между строками.
- `first_column_offset_x` — смещение первой колонки относительно блока месяца.
- `second_column_offset_x` — смещение второй колонки относительно блока месяца.
- `month_blocks` — координаты блоков месяцев.

Координаты считаются от левого верхнего угла изображения.

## Редактирование настроек через интерфейс

Настройки экспорта можно редактировать из главного окна приложения.

Доступные действия:

- открыть существующий JSON настроек;
- создать настройки по умолчанию;
- изменить размер изображения;
- изменить фон, шаблон, шрифт и цвет текста;
- изменить координаты блоков месяцев;
- сдвинуть все месяцы сразу;
- скопировать координаты одного месяца в другой;
- сохранить настройки;
- сохранить настройки как новый JSON.

### Координаты месяцев

Каждый месяц имеет координаты `x` и `y` в блоке `month_blocks`.

Например:

```json
"month_blocks": {
  "1": {"x": 40, "y": 60},
  "2": {"x": 360, "y": 60}
}
```

Координаты считаются от левого верхнего угла изображения.

### Сдвиг всех месяцев

Функция сдвига всех месяцев изменяет координаты всех блоков сразу.

Например, если указать:

- Сдвиг X: 10
- Сдвиг Y: 20

то каждый месяц будет сдвинут на 10 пикселей вправо и на 20 пикселей вниз.

### Копирование координат месяца

Можно скопировать координаты одного месяца в другой.

Например:

- Копировать из месяца: 1
- Копировать в месяц: 2

После копирования месяц 2 получит такие же `x` и `y`, как месяц 1.

### Предпросмотр

В редакторе можно открыть предпросмотр изображения для выбранного города.
Если город не выбран или города не загружены, предпросмотр будет недоступен.
