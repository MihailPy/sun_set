# Sun Set

Sun Set — настольное приложение для расчёта, просмотра,
редактирования и экспорта данных о времени заката для нескольких городов.

Поддерживаемые системы:

- macOS;
- Windows.

## Возможности

- создание проектов с несколькими городами;
- расчёт времени заката на выбранный год;
- выбор двух интересующих дней недели;
- ручное редактирование рассчитанных данных;
- контроль актуальности данных после изменения параметров города;
- предпросмотр и экспорт расписаний в PNG;
- настройка внешнего вида экспортируемых изображений;
- поддержка старого формата файлов со списком городов.

## Стек

- Python `3.13+`
- `PyQt6`
- `astral`
- `dacite`
- `pytest` + `pytest-qt`
- `uv` (рекомендуемый менеджер окружения и запуска)

## Быстрый старт

### 1) Установка зависимостей

```bash
uv sync
```

### 2) Запуск приложения

```bash
uv run start-app
```


## Установка готового приложения

Готовые сборки находятся в разделе Releases:

- `SunSet-macOS.zip` — версия для macOS;
- `SunSet-Windows.zip` — версия для Windows.

### macOS

1. Распакуйте архив.
2. Переместите `SunSet.app` в папку `Applications`.
3. Запустите приложение.

Поскольку первая версия приложения не подписана сертификатом Apple,
macOS может потребовать подтверждение запуска через:

`Системные настройки → Конфиденциальность и безопасность`.

### Windows

1. Распакуйте архив.
2. Запустите `SunSet.exe` из папки `SunSet`.

Готовые сборки будут опубликованы в разделе Releases.

## Основной сценарий работы

1. Создайте новый проект или откройте существующий JSON-файл.
2. Добавьте города и заполните их параметры.
3. Выберите нужные города флажками в таблице.
4. Укажите год и два дня недели.
5. Нажмите кнопку обновления данных заката.
6. Проверьте значение в колонке `Статус`.
7. Для просмотра данных нажмите `Открыть`
   в колонке `Данные заката`.
8. При необходимости отредактируйте отдельные значения.
9. Сохраните проект.

## Статусы данных

- **Актуально** — данные соответствуют текущим параметрам города.
- **Требует обновления** — координаты, часовой пояс или высота изменились.
- **Изменено вручную** — отдельные значения заката были отредактированы.
- **Ошибка расчёта** — расчёт не завершён из-за полярных условий.
- **Нет данных** — данные заката ещё не рассчитаны.

## Экспорт изображений

Приложение поддерживает экспорт расписания захода солнца в PNG.
Данные размещаются на белом фоне или поверх шаблона по JSON-настройкам.

- [Формат настроек экспорта изображения](docs/image_export_settings.md)

## Формат проекта

Новый формат проекта представляет собой JSON-объект,
который содержит:

- версию формата;
- настройки года и дней недели;
- список городов;
- пути настроек экспорта;
- каталог экспорта.

Старый формат в виде массива городов по-прежнему поддерживается
и автоматически преобразуется при последующем сохранении.

## Структура проекта

```text
src/sun_set/
  constants/     # значения по умолчанию и формат проекта
  core/          # расчёт данных заката
  image_export/  # экспорт изображений и настройки
  models/        # модели данных и Qt table model
  services/      # сценарии приложения и workflow
  storage/       # чтение и запись файлов
  views/         # окна и диалоги PyQt6
  main.py        # точка входа

docs/            # пользовательская и техническая документация
examples/        # примеры настроек
scripts/         # сборка приложения и генерация ресурсов
tests/           # unit-, UI- и интеграционные тесты
```

## Точка входа

CLI-скрипт зарегистрирован в `pyproject.toml`:

```toml
[project.scripts]
start-app = "sun_set.main:main"
```
